วิธีลง GitHub Pages

ขั้นตอนการอัพโหลดเว็บสตรีมมิ่งช่องกีฬาขึ้น GitHub Pages

ขั้นตอนการลง GitHub Pages

1

สร้างบัญชี GitHub

ไปที่ github.com และสมัครบัญชีใหม่ (หากยังไม่มี)

2

สร้าง Repository ใหม่

คลิกปุ่ม "+" มุมขวาบน และเลือก "New repository"

ตั้งชื่อ repository เช่น sports-streaming

เลือก "Public" และสร้าง repository

3

อัพโหลดไฟล์

ใน repository ที่สร้างใหม่ ให้คลิก "Add file" → "Upload files"

ลากไฟล์ index.html ที่มีโค้ดเว็บสตรีมมิ่งของคุณมาใส่ในนี้

4

ตั้งค่า GitHub Pages

ไปที่แท็บ "Settings" → "Pages"

ที่ส่วน "Branch" ให้เลือก "main" และกด Save

5

รอการ deploy

รอสักครู่ (ไม่เกิน 2-3 นาที) แล้ว刷新หน้า

คุณจะเห็นลิงก์ไปยังเว็บไซต์ของคุณ เช่น https://username.github.io/sports-streaming

README.md 
# Sports Streaming Website

เว็บสตรีมมิ่งช่องกีฬาด้วย HTML, CSS และ JavaScript

## คุณสมบัติ

- แสดงช่องสตรีมมิ่งกีฬาหลากหลายประเภท
- คลิกเพื่อเปิดใน WisePlay โดยอัตโนมัติ
- คลิกขวาเพื่อคัดลอก URL สตรีม
- การค้นหาช่อง
- มุมมองแบบกริดและการ์ด

## วิธีการใช้งาน

1. คลิกที่โลโก้ช่องเพื่อเปิดใน WisePlay
2. คลิกขวาที่โลโก้ช่องเพื่อคัดลอก URL
3. ใช้ช่องค้นหาเพื่อค้นหาช่องที่ต้องการ

## การติดตั้งบน GitHub Pages

1. อัพโหลดไฟล์ index.html ไปยัง repository
2. ไปที่ Settings → Pages
3. เลือก Branch หลัก (main/master)
4. รอการ deploy และเข้าใช้งานที่ https://username.github.io/repository-name

หมายเหตุสำคัญ

• GitHub Pages ใช้ได้กับ static content เท่านั้น (HTML, CSS, JavaScript)

• ไฟล์หลักต้องชื่อ index.html ถึงจะทำงานได้ถูกต้อง

• การเปลี่ยนแปลงอาจใช้เวลา 1-2 นาทีในการอัพเดทบน GitHub Pages

• หากมีปัญหา ให้ตรวจสอบ Console (F12) เพื่อดูข้อผิดพลาด
