ชมการถ่ายทอดสดกีฬาทุกประเภทจากทุกมุมโลก

คำแนะนำ: คลิกที่โลโก้ช่องเพื่อเปิดใน WisePlay หรือคลิกขวาเพื่อคัดลอกลิงก์สตรีม

คำแนะนำการใช้งานกับ GitHub Pages

1. สร้าง repository ใหม่บน GitHub

2. อัพโหลดไฟล์นี้เป็น index.html

3. ไปที่ Settings → Pages → แบรนช์เลือก main หรือ master → Save

4. รอสักครู่แล้วเว็บของคุณจะพร้อมใช้งานที่ https://[username].github.io/[repository-name]
